## Loading the mod

to load the mod, click the green `Code` button, and click `Download ZIP` to download the mod. **you may need [7-Zip](https://www.7-zip.org/a/7z2600-x64.exe) or [WinRAR](https://www.win-rar.com/fileadmin/winrar-versions/downloader/WinRAR-720.exe) to extract the mod `.zip` file.** 

now go to your Modern Warfare Folder, and right click and create a new **mods** folder. inside of this folder, you will extract the zip folder and rename it to `angora`. if done right, the structure of the folder should look like this:
```
Modern Warfare Remastered
├── main
├── raw
├── mods *(you create this folder)*
│   ├── angora
│   │   ├── scripts
|   |   |   └── ...
└── h1-mod.exe
```
